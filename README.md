# HookedAI

Turn any photo into a crochet pattern — powered by AI.

HookedAI is a web app that lets crocheters upload a photo and receive a stitch-by-stitch grid pattern. It uses computer vision to analyze images, extract colors, and generate patterns based on user preferences like yarn weight, hook size, skill level, and grid dimensions.

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

|-----------------|--------------------------|
